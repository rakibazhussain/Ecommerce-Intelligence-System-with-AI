## 📌 Overview
This project is an end-to-end AI-powered system designed to forecast product sales and provide real-time business insights. It combines machine learning, real-time data streaming, and interactive dashboards to enable proactive decision-making.

---

## 🎯 Problem Statement
Retail businesses often rely on historical reports, which delay decision-making. There is a need for a system that:
- Predicts future sales trends  
- Processes real-time data  
- Alerts stakeholders instantly  

---

## 💡 Solution
This system integrates machine learning with a real-time data pipeline to:
- Forecast sales trends using historical data  
- Continuously update insights using streaming data  
- Provide live dashboards for business monitoring  
- Send instant alerts for significant sales changes  

---

## 🧠 Key Features
- Machine Learning-based sales forecasting  
- Real-time data streaming using Apache Kafka  
- Interactive Power BI dashboards  
- Automated SMS alerts using Twilio API  
- PostgreSQL database for structured storage  

---

## 🏗️ System Architecture
Data Source → Kafka → PostgreSQL → ML Model → Power BI → Alerts (Twilio)

---

## ⚙️ Tech Stack
- Programming: Python  
- Machine Learning: Scikit-learn  
- Data Streaming: Apache Kafka  
- Database: PostgreSQL  
- Visualization: Power BI  
- Alerts: Twilio API  

---

## 📊 Results & Impact
- Achieved ~85–90% forecasting accuracy  
- Enabled real-time monitoring of sales trends  
- Reduced delay in business decision-making  
- Improved visibility into high and low performing products

## 🔄 Workflow
1. Collect historical sales data  
2. Stream incoming data using Kafka  
3. Store and process data in PostgreSQL  
4. Train and apply ML model for predictions  
5. Visualize insights in Power BI  
6. Trigger alerts using Twilio  

---
